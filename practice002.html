Hypertext Markup Language (HTLM) is the standard Markup language for creating web pages and web applications.Web browers receive HTML documents from a web server or from local strong and render them into multimedia web pages. HTMl describes the structure of a web page sematically and orginally included cues for the appearance of the document.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page.
